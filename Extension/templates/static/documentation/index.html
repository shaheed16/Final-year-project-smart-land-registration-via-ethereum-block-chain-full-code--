Getting Started

Setting Up The Dev Tools

To get started with Propertier Landing Page Template as a starter template you need to intialize the basic bootstrap and js files and main stylesheet of the theme.

Copy to clipboard 
    
<link rel="stylesheet" href="assets/bootstrap/css/bootstrap.min.css">
<script src="libs/jquery/jquery.min.js"></script>
<script src="assets/bootstrap/js/bootstrap.min.js"></script>
<!-- CSS -->
<link rel="stylesheet" type="text/css" href="style.css"/>

Use .container-fluid for a full width container, spanning the entire width of the viewport.

Copy to clipboard 
    
<div class="container-fluid">...</div>

Animation On Scroll(AOS)

To initialize AOS library you need a stylesheet at the head section and two scripts at the bottom of the page first is the main js file of AOS and second is the initialization file of AOS which you must require inorder to intialize the library

AOS Stylesheet
Copy to clipboard 
    
<link href="libs/aos/aos-master/dist/aos.css" rel="stylesheet">
AOS Scripts
Copy to clipboard 
    
<script src="libs/aos/aos-master/dist/aos.js"></script>
<script src="libs/aos/aos-init.js"></script>

Slick Slider

To initialize Slick Slider you need to add these scripts and stylesheets at the head section.

Copy to clipboard 
    
<link rel="stylesheet" type="text/css" href="assets/slick-slider/slick/slick.css"/>
<link rel="stylesheet" type="text/css" href="assets/slick-slider/slick/slick-theme.css"/>
<script type="text/javascript" src="assets/slick-slider/slick/slick.min.js"></script>

<!-- Init Slick Slider -->
<script type="text/javascript" src="assets/slick-slider/slick-init.js"></script>

You must add slick-init.js to initialize the slider.

Font Awesome

To initialize Font Awesome you need to add these scripts and stylesheets at the head section.

Copy to clipboard 
    

<link rel="stylesheet" href="libs/font-awesome/css/all.min.css">
<script src="libs/font-awesome/js/all.min.js"></script>

Assets

Assets folder contains all of the assets referened in htm pages.

images folder contains folder names of images as portfolios, team etc. in which respective images are placed.

Bootstrap css and js files are included in bootstrap folder, bootstrap scss files are also included in bootstrap folder with predefined variables.

Libs

The libs folder include libraries like aos,font-awesome,jquery and zoom-master.

CSS

The main stylesheet of the theme is included in the main folder of Propertier Landing Page Template along with html pages while external stylesheets are present inside css folder.

Other stylesheets of frameworks and libraries that are included in assets and libs folder respectively.

JS

JS files used in theme are included in js folder where mostly initialization scripts are placed for different functionality of the theme.

Libs

The libs folder include libraries like aos,font-awesome,jquery and zoom-master.

Back To Top Button

The Back To Top button allows the user to scroll to the top of the page easily without having to scroll all the way through.

How to add the button

By default this button is added to all of the homepages but if you want to add it to a page then you can use the following html and place it below the body tag.

Copy to clipboard 
           

<!-- Back to top button -->
<a id="backtotop"></a>

Change Button Color

For main homepage the Back To Top button css is present inside the main stylesheet at the very bottom of the stylesheet while for homepages its corresponding css is present inside their external stylesheets inside css folders also at the very bottom of the stylesheet.

Copy to clipboard 
           

#backtotop {
  display: inline-block;
  background-color: #28a745;
  width: 50px;
  height: 50px;
  text-align: center;
  border-radius: 4px;
  position: fixed;
  bottom: 30px;
  right: 30px;
  transition: background-color .3s, 
    opacity .5s, visibility .5s;
  opacity: 0;
  visibility: hidden;
  z-index: 1000;
}

Navbars

Propertier Landing Page Template contains two navbars styles which are plain and transparent. Among 6 homepages both plain and transparent navbars are used.

Transparent Navbar

By default plain navbar is activated but you can easily switch to transparent navbar.

Note! Inorder to use transparent navbar you must use a homepage that contains transparent navbar eg: index-2.html

Remove Transparent

To remove the transparency of the navbar you can simply add a class bgp to the <header> and <nav> element of the page which will make the navbar plain.

Sticky Navbar

By default all navbars in Propertier Landing Page Template are sticky but if you don't want a sticky navbar you can simple remove it by adding a class non-sticky to the <nav> element.

Copy to clipboard 
           
           
<nav class="navbar navbar-expand-lg navbar-light cs-nav non-sticky" id="navbar">

...

</nav>

Google Maps

Google maps are used for all homepages and contact pages for easy location sharing.

Propertier Landing Page Template uses iframe for google map integration where use can use the following code to integrate google map and add a location to src attribute to load your current map.

Copy to clipboard 
      
           
            
<iframe src="" width="600" height="450" frameborder="0" style="border:0;" allowfullscreen="" aria-hidden="false" tabindex="0"></iframe>

Map Attributes

The iframe code above uses some attributes which you can use to customize the map which are.

  • width for custom width as width="600" or for full width use can use width="100%"
  • Height for height as height="450"
  • allowfullscreen to enable full screen and other attributes which you can leave as default.

Add Map Location

Inorder to add a location to the map you can visit https://www.google.com/maps and click on the location from the map which will bring a map marker to that location then at the top left corner you can click the share button which will open a modal window then click on embed map and code the iframe code and paste it on the page where you want to add the map.

Note! For map zoom you can use zoom the map on google maps then embed it using the same process which will embed the zoomed version of the map.
